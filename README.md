Employee Management System (EMS)
A secure and scalable employee management application built with Java, Spring Boot, Spring Security, JWT, and SQL. The system provides role-based access control, authentication, and CRUD operations for managing employee records efficiently.

Features
-> Secure authentication and authorization with Spring Security + JWT

-> Role-based access (Admin, HR, Employee)

-> CRUD operations for employee records

-> Department and role management

-> Password hashing and token-based session management

-> RESTful APIs with layered architecture (Controller → Service → Repository → Entity)

-> Input validation, exception handling, and logging
